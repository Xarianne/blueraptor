# Bluewhale

A personal image based on the [finpilot](https://github.com/projectbluefin/finpilot) template by Universal Blue. 

Has Hyprland with Dank Material Shell (home dots managed by Distrobox) as well as GNOME. Terra repos enabled for codecs and latest mesa. I will happily break this to experiment with things. Might not work for you.
